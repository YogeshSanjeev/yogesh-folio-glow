

# Yogesh Kumar — Personal Portfolio Website

## Overview
A modern, minimal personal portfolio website with clean typography, generous whitespace, subtle animations, and soft neutral colors with minimal accent highlights. Single-page scrolling layout with smooth navigation.

---

## Pages & Sections

### 1. Navigation Bar
- Fixed top navbar with name/logo on the left
- Smooth-scroll links: Home, About, Education, Experience, Skills, Services, Contact
- Minimal hamburger menu on mobile

### 2. Hero Section (Home)
- Large headline: **"Yogesh Kumar — Full-Stack Web Developer"**
- Subheadline and short intro paragraph from the provided bio
- Circular profile photo placeholder (user can replace later)
- Two CTA buttons: **View Projects** and **Contact Me**
- Row of tech badges: React, Node.js, Python, MongoDB, REST APIs
- Subtle fade-in animations on load

### 3. About Section
- 2–3 paragraph expanded bio highlighting full-stack focus, backend interests, continuous learning, and problem-solving
- "Currently" highlight card showing: **Software Developer Intern at Knewton Tech (Jan 2026 – Present)**

### 4. Education Section

  - MCA — Adhiyamaan College of Engineering — Expected 2026
  - BCA — Arignar Anna College — 2023

  - Secondary — SM English High School — 2018

### 5. Experience Section
- Card layout for the Knewton Tech internship with bullet points covering responsibilities (development, clean code, collaboration, bug fixing, testing & deployment)

### 6. Skills Section
- Categorized grid with icons for each skill group:
  - Programming Languages (Python, JavaScript)

### 7. Services Section

### 8. Contact / Footer
- Simple contact section with email link and social placeholders

## Design & Style
- **Colors**: Soft neutral palette (whites, light grays, dark charcoal text) with one subtle accent color
- **Typography**: Strong, modern font hierarchy with generous spacing
- **Layout**: Clean sections with ample whitespace, no clutter
- **Animations**: Subtle fade-in/slide-up on scroll using CSS transitions
- **Responsive**: Fully mobile-friendly layout
- **No backend needed** — purely static frontend
